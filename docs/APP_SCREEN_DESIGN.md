# 텅장히어로 앱 화면 설계서

## 개요

본 문서는 "텅장히어로" 앱의 화면 구성과 UI/UX 설계를 정의한다.

### 설계 원칙

1. **이중 앱 통합**: 가계부와 게임이 자연스럽게 연결
2. **직관적 네비게이션**: 핵심 기능에 2탭 이내 접근
3. **시각적 피드백**: 모든 행동에 즉각적 반응
4. **일관된 디자인 언어**: 통일된 컬러/폰트/아이콘 체계

### 컬러 팔레트

| 용도 | 색상 | Hex |
|------|------|-----|
| Primary | 골드/황금 | #FFD700 |

| Success | 그린 | #4CAF50 |

| 요소 | 애니메이션 |
|------|-----------|

| 토글 | 스프링 애니메이션 |

| 이펙트 | 설명 |
|--------|------|

| 레벨업 | 전체 화면 플래시 + 파티클 |

| 강화 성공 | 빛 폭발 + 진동 |

### 11.1 화면 크기 대응

| 화면 | 작은 폰 (< 375px) | 표준 (375-414px) | 큰 폰 (> 414px) |

### 11.2 노치/다이나믹 아일랜드 대응

- 상단 Safe Area 준수
- 하단 홈 인디케이터 영역 확보

### 11.3 가로 모드

- 기본: 지원하지 않음 (세로 고정)
- 선택적: 통계 차트 화면만 가로 지원
