# 텅장히어로 RPG 시스템 설계서

## 개요

본 문서는 가계부 앱 "텅장히어로"의 RPG 시스템 설계를 정의한다.

### 핵심 철학

**가계부와 게임의 상호 의존**

- 가계부: 게임에서 얻기 어려운 핵심 재화 획득
- 게임: 방치형 자동 사냥으로 기본 재화 획득
- 둘 다 해야 캐릭터가 온전히 성장

### 시스템 구조

```
[가계부 영역]
거래 기록 / 퀘스트 완료
        |
        v
핵심 재화 획득
(강화석, 스킬북, 가챠 티켓, 각성석 등)
        |
        +-- 일반 기록: 기본 보상 (일일 상한)
        +-- 영수증 인증: 추가 보상 (상한 증가)


[게임 영역]
방치형 자동 사냥 (24시간)
        |
        v
기본 재화 획득
(골드, 경험치, 일반 장비)
```

### 성장 구조

| 요소 | 획득처 | 용도 |
|------|--------|------|
| 골드 | 게임 (사냥) | 상점 구매, 기본 소비 |
| 경험치 | 게임 (사냥) | 레벨업 |
| 일반 장비 | 게임 (드롭) | 착용, 분해 |
| 강화석 | 가계부 | 장비 강화 |
| 스킬북 | 가계부 | 공용 스킬 습득 |
| 가챠 티켓 | 가계부 | 고급 장비/펫 뽑기 |
| 각성석 | 가계부 | 장비 각성 |
| 프리셋 슬롯 | 프리미엄 | 스탯/스킬 프리셋 확장 |

## 1. 가계부 보상 시스템

### 1.1 보상 재화 종류

| 재화 | 용도 | 희귀도 |
|------|------|--------|
| 강화석 | 장비 강화 | 일반 |
| 고급 강화석 | 고강화 (+7 이상) | 희귀 |
| 스킬북 | 공용 스킬 습득/강화 | 일반~희귀 |
| 가챠 티켓 | 일반 가챠 | 일반 |
| 프리미엄 티켓 | 고급 가챠 | 희귀 |
| 각성석 | 장비 각성 | 희귀 |
| 펫 먹이 | 펫 성장 | 일반 |

### 1.2 기본 보상 (영수증 없음)

일일 상한 내에서 기록당 랜덤 보상 지급

#### 기록당 보상 (택1 랜덤)
기록 1건당 아래 보상 중 **하나**를 랜덤으로 지급

| 보상 | 확률 | 개수 |
|------|------|------|
| 강화석 | 60% | 1~2 |
| 스킬북 (일반) | 20% | 1 |
| 가챠 티켓 | 10% | 1 |
| 펫 먹이 | 10% | 1~3 |

**합계: 100%** (반드시 하나의 보상 지급)

#### 일일 상한
| 항목 | 상한 |
|------|------|
| 보상 받을 수 있는 기록 수 | 10건 |
| 강화석 | 10개 |
| 스킬북 | 2개 |
| 가챠 티켓 | 1개 |

#### 기록 조건
- 최소 금액: 1,000원 이상
- 카테고리 선택 필수
- 동일 금액 연속 기록 제한 (3회까지)

### 1.3 영수증 인증 보상

영수증 사진 첨부 시 추가 보상 및 상한 증가

#### 인증 보상 (기본 보상 외 추가, 택1 랜덤)
영수증 인증 시 기본 보상과 별도로 아래 보상 중 **하나**를 추가 지급

| 보상 | 확률 | 개수 |
|------|------|------|
| 강화석 | 50% | 2~3 |
| 고급 강화석 | 20% | 1 |
| 스킬북 (고급) | 15% | 1 |
| 가챠 티켓 | 10% | 1 |
| 프리미엄 티켓 | 5% | 1 |

**합계: 100%** (인증 시 반드시 추가 보상 1개 지급)

#### 인증 시 상한 증가
| 항목 | 기본 상한 | 인증 포함 상한 |
|------|-----------|----------------|
| 보상 기록 수 | 10건 | 20건 |
| 강화석 | 10개 | 25개 |
| 스킬북 | 2개 | 5개 |
| 가챠 티켓 | 1개 | 3개 |

#### 영수증 인증 방식
1. 거래 기록 시 영수증 사진 첨부 (선택)
2. OCR로 금액/날짜/가맹점 추출
3. 입력 내용과 대조 검증
4. 일치 시 인증 완료, 추가 보상 지급

#### 인증 실패 케이스
- 금액 불일치 (10% 이상 차이)
- 날짜 불일치 (당일/전일 아님)
- 이미지 인식 실패 (수동 검토 요청)
- 동일 영수증 중복 사용

### 1.4 퀘스트 보상

#### 일일 퀘스트 (매일 3개)
| 퀘스트 | 조건 | 보상 |
|--------|------|------|
| 오늘의 기록 | 거래 1건 기록 | 강화석 3 |
| 성실한 기록 | 거래 5건 기록 | 강화석 5, 스킬북 1 |
| 인증왕 | 영수증 인증 3건 | 가챠 티켓 1, 고급 강화석 2 |
| 알뜰한 하루 | 일일 지출 3만원 이하 | 강화석 5 |
| 무지출 도전 | 오늘 지출 0원 | 프리미엄 티켓 1 |
| 저축 기록 | 수입 기록 1건 | 강화석 3, 펫 먹이 5 |

#### 주간 퀘스트 (매주 5개)
| 퀘스트 | 조건 | 보상 |
|--------|------|------|
| 7일 연속 기록 | 7일 연속 기록 | 가챠 티켓 3, 스킬북 (고급) 1 |
| 인증 마스터 | 주간 영수증 인증 20건 | 프리미엄 티켓 2, 각성석 1 |
| 예산 수호자 | 주간 예산 준수 | 가챠 티켓 2, 고급 강화석 5 |
| 절약 달인 | 주간 지출 20만원 이하 | 강화석 20 |
| 저축왕 | 주간 저축 10만원 이상 | 프리미엄 티켓 1, 스킬북 (고급) 2 |

#### 월간 퀘스트
| 퀘스트 | 조건 | 보상 |
|--------|------|------|
| 30일 연속 기록 | 30일 연속 기록 | 각성석 2, 프리미엄 티켓 5 |
| 인증 달인 | 월간 영수증 인증 50건 | 각성석 3, 프리미엄 티켓 3 |
| 예산의 제왕 | 월간 예산 준수 | 스킬북 (희귀) 1, 각성석 2 |
| 백만 저축 | 월 저축 100만원 | 프리미엄 티켓 10 |

### 1.5 연속 기록 보너스

| 연속 일수 | 보상 |
|-----------|------|
| 3일 | 강화석 5 |
| 7일 | 가챠 티켓 1 |
| 14일 | 스킬북 (고급) 1, 강화석 10 |
| 21일 | 프리미엄 티켓 1 |
| 30일 | 각성석 1, 가챠 티켓 3 |
| 60일 | 스킬북 (희귀) 1, 각성석 2 |
| 100일 | 프리미엄 티켓 5, 각성석 3 |
| 365일 | 특별 칭호, 전설 장비 선택권 |

#### 연속 기록 보호 시스템

##### 연속 기록 보호권
- 연속 기록이 끊기기 전에 사용하여 1일 보호
- 자정까지 기록하지 못한 경우 자동 소진
- 획득 방법:
  - 7일 연속 기록 시 1개 지급
  - 30일 연속 기록 시 2개 지급
  - 업적/이벤트 보상

##### 연속 기록 복구권 (희귀)
- 끊긴 연속 기록을 최대 3일 전까지 복구
- 획득 방법:
  - 프리미엄 상점 (유료)
  - 특별 이벤트 보상
  - 365일 달성 시 1개 지급

##### 제한 사항
- 보호권은 최대 3개까지 보유 가능
- 복구권은 동일 연속 기록에 1회만 사용 가능

## 2. 챌린지 퀘스트 시스템

### 2.1 개요

챌린지 퀘스트는 예산 절약 목표를 게임화한 시스템이다.
솔로 모드에서는 시스템이 랜덤하게 제공하고,
커뮤니티 모드에서는 유저가 직접 만들어 다른 유저와 함께 도전한다.

### 2.2 솔로 챌린지 (시스템 제공)

#### 개요
- 시스템이 랜덤하게 챌린지 제공
- 일일/주간/월간 단위로 다양한 목표
- 달성 시 재화 보상

#### 챌린지 풀 (예시)

##### 일일 챌린지
| 챌린지 | 조건 | 보상 |
|--------|------|------|
| 오늘 커피 안 사먹기 | 카페 지출 0원 | 강화석 5 |
| 점심 만원 이하 | 식비(점심) 10,000원 이하 | 강화석 3 |
| 교통비 절약 | 교통 지출 5,000원 이하 | 강화석 3 |
| 무지출 도전 | 오늘 총 지출 0원 | 강화석 10, 스킬북 1 |
| 현금 없는 하루 | 현금 지출 0원 | 강화석 3 |

##### 주간 챌린지
| 챌린지 | 조건 | 보상 |
|--------|------|------|
| 식비 15만원 이하 | 주간 식비 150,000원 이하 | 강화석 20, 스킬북 1 |
| 배달 3회 이하 | 주간 배달 음식 3회 이하 | 강화석 15 |
| 쇼핑 5만원 이하 | 주간 쇼핑 50,000원 이하 | 강화석 15, 가챠 티켓 1 |
| 문화비 2만원 이하 | 주간 문화/여가 20,000원 이하 | 강화석 10 |
| 주말 무지출 | 토, 일 지출 0원 | 강화석 25, 스킬북 2 |

##### 월간 챌린지
| 챌린지 | 조건 | 보상 |
|--------|------|------|
| 식비 50만원 이하 | 월간 식비 500,000원 이하 | 각성석 1, 강화석 50 |
| 총 지출 100만원 이하 | 월간 총 지출 1,000,000원 이하 | 각성석 2, 스킬북 (희귀) 1 |
| 저축 30만원 이상 | 월간 저축 300,000원 이상 | 각성석 2, 프리미엄 티켓 3 |
| 고정비 제외 50만원 이하 | 변동 지출 500,000원 이하 | 각성석 1, 가챠 티켓 3 |

#### 챌린지 제공 방식
- 일일: 매일 자정에 2~3개 랜덤 제공
- 주간: 매주 월요일에 2개 랜덤 제공
- 월간: 매월 1일에 1개 랜덤 제공
- 유저 소비 패턴 분석 후 적절한 난이도 제공

#### 난이도 조절
유저의 평균 지출 패턴을 분석하여 개인화된 목표 제시

```
예시:
- 평균 주간 식비 20만원인 유저
  -> "이번 주 식비 18만원 이하" 챌린지 제공

- 평균 주간 식비 10만원인 유저
  -> "이번 주 식비 8만원 이하" 챌린지 제공
```

### 2.3 커뮤니티 챌린지 (유저 생성)

#### 개요
- 유저가 직접 챌린지를 생성하여 공유
- 다른 유저들이 참여 신청
- 함께 도전하고 달성 시 모두 보상

#### 챌린지 생성

##### 생성 가능 항목
| 항목 | 설정 가능 값 |
|------|--------------|
| 대상 카테고리 | 전체, 식비, 교통, 쇼핑, 문화 등 |
| 목표 유형 | 금액 이하, 횟수 이하, 금액 0원 |
| 목표 값 | 자유 설정 |
| 기간 | 1일, 3일, 7일, 14일, 30일 |
| 참가 인원 | 최소 2명 ~ 최대 50명 |

##### 생성 예시
```
챌린지명: "이번 주 카페 2만원 챌린지"
카테고리: 카페/음료
목표: 20,000원 이하
기간: 7일
참가 인원: 10명
보상: 강화석 15 (달성 시)
```

#### 참여 방식
1. 커뮤니티에서 챌린지 목록 확인
2. 원하는 챌린지에 참여 신청
3. 챌린지 시작 시간에 자동 시작
4. 기간 동안 해당 카테고리 지출 추적
5. 종료 시 결과 확인 및 보상 지급

#### 보상 구조

##### 기본 보상
챌린지 달성 시 보상 지급

| 기간 | 기본 보상 |
|------|-----------|
| 1일 | 강화석 3~5 |
| 3일 | 강화석 8~12 |
| 7일 | 강화석 15~25 |
| 14일 | 강화석 30~50, 스킬북 1 |
| 30일 | 강화석 50~100, 각성석 1 |

##### 추가 보상 (커뮤니티)
| 조건 | 추가 보상 |
|------|-----------|
| 전원 달성 | 보상 +50% |
| 달성률 80% 이상 | 보상 +20% |
| 연속 참여 3회 | 강화석 +10 |
| 챌린지 생성자 | 참여자 수 x 강화석 1 |

#### 커뮤니티 기능

##### 챌린지 피드
- 진행 중인 챌린지 목록
- 인기 챌린지 (참여자 많은 순)
- 신규 챌린지
- 친구가 참여 중인 챌린지

##### 챌린지 내 소통
- 참여자 간 응원 메시지
- 진행 상황 공유 (선택적)
- 달성 시 축하 알림

##### 랭킹
- 챌린지 달성률 랭킹
- 연속 달성 랭킹
- 챌린지 생성 랭킹

#### 어뷰징 방지 시스템

##### 챌린지 생성 제한
| 제한 항목 | 조건 |
|-----------|------|
| 생성 자격 | 계정 생성 후 7일 경과 + 기록 20건 이상 |
| 일일 생성 제한 | 최대 2개 |
| 동시 운영 제한 | 최대 5개 |
| 목표 금액 최소값 | 카테고리별 평균 지출의 50% 이상 |
| 기간 제한 | 최소 1일, 최대 30일 |

##### 참여 제한
| 제한 항목 | 조건 |
|-----------|------|
| 참여 자격 | 계정 생성 후 3일 경과 + 기록 5건 이상 |
| 동시 참여 제한 | 최대 10개 |
| 자기 챌린지 참여 | 불가 (생성자는 자동 참여) |
| 탈퇴 후 재참여 | 동일 챌린지 재참여 불가 |

##### 다계정 감지
| 감지 항목 | 조치 |
|-----------|------|
| 동일 기기 복수 계정 | 경고 후 챌린지 기능 제한 |
| 동일 IP 의심 패턴 | 모니터링 대상 등록 |
| 영수증 중복 사용 | 해당 기록 무효화 + 챌린지 실패 처리 |
| 비정상 거래 패턴 | 수동 검토 후 조치 |

##### 챌린지 검증
- 참가 인원 최소 2명 미달 시 자동 취소
- 생성자 탈퇴 시 챌린지 유지 (보상 지급)
- 신고 3건 이상 시 운영진 검토
- 악용 확인 시 생성자 챌린지 기능 30일 정지

##### 보상 제한
| 상황 | 조치 |
|------|------|
| 참여자 1명 (생성자만) | 보상 50% 감소 |
| 목표 달성률 100% 연속 5회 | 난이도 하향 조정 권고 |
| 동일 유저 간 반복 참여 | 3회차부터 보상 30% 감소 |

### 2.4 챌린지 데이터 구조

```
Challenge {
```

| 100 | 398,107 | ~20,000,000 | 약 30일 |

*소요 시간은 부스터, 장비 옵션 미적용 기준*

## 5. 스탯 프리셋 시스템

직업 대신 자유로운 스탯 배분과 프리셋 저장으로 다양한 빌드 구성

### 5.1 스탯 초기화

- **무료 초기화**: 언제든지 스탯 포인트 전체 회수 가능
- 초기화 시 모든 배분된 포인트가 미배분 상태로 복귀
- 초기화 쿨타임: 없음 (자유롭게 변경 가능)

### 5.2 프리셋 슬롯

스탯 배분 + 공용 스킬 세팅을 저장하는 프리셋

| 구분 | 슬롯 수 | 획득 방법 |
|------|---------|-----------|
| 기본 제공 | 3개 | 게임 시작 시 |
| 최대 확장 | 5개 | 프리미엄 구매 (1슬롯당 1,900원) |

#### 프리셋에 저장되는 정보
- 스탯 포인트 배분 상태
- 공용 스킬 슬롯 배치 (2개)
- 프리셋 이름 (사용자 지정)
- 프리셋 아이콘 (선택)

#### 프리셋 전환
- 전투 중이 아닐 때 즉시 전환 가능
- 전환 시 스탯/스킬이 해당 프리셋으로 변경
- 장비는 프리셋에 포함되지 않음 (별도 관리)

### 5.3 추천 빌드 템플릿

게임 시작 시 참고할 수 있는 기본 빌드 가이드

#### 물리 공격형
```
추천 스탯: ATK 40% / HP 30% / SPD 20% / LUK 10%
특징: 높은 물리 데미지, 크리티컬 의존
추천 무기: 검, 대검, 단검
```

#### 마법 공격형
```
추천 스탯: MAG 45% / MP 25% / MDF 20% / SPD 10%
특징: 높은 마법 데미지, 광역기 활용
추천 무기: 지팡이, 완드
```

#### 탱커형
```
추천 스탯: HP 40% / DEF 35% / MDF 15% / ATK 10%
특징: 높은 생존력, 안정적 사냥
추천 무기: 검, 철퇴 + 방패
```

#### 밸런스형
```
추천 스탯: HP 25% / ATK 25% / DEF 20% / SPD 15% / LUK 15%
특징: 균형 잡힌 성능, 초보자 추천
추천 무기: 모든 무기 사용 가능
```

### 5.4 프리셋 활용 예시

| 프리셋명 | 용도 | 스탯 방향 |
|----------|------|-----------|
| 사냥용 | 일반 스테이지 | SPD/LUK 중심 (빠른 사냥) |
| 보스용 | 주간 보스 | HP/DEF 중심 (생존 + 딜) |
| 골드 파밍 | 재화 수집 | LUK 최대 (드롭률 증가) |

## 6. 스킬 시스템 (하이브리드)

장비 고유 스킬 + 공용 스킬 슬롯의 조합

### 6.1 스킬 구조 개요

```
┌─────────────────────────────────────┐
│           전투 스킬 구성             │
├─────────────────────────────────────┤
│                                     │
│  [장비 스킬] - 장비에 귀속           │
│  ├─ 무기 스킬 (액티브 1개)          │
│  ├─ 갑옷 스킬 (패시브 1개)          │
│  └─ 장신구 스킬 (패시브 1개)        │
│                                     │
│  [공용 스킬] - 프리셋에 저장         │
│  └─ 공용 슬롯 2개 (스킬북으로 습득)  │
│                                     │
└─────────────────────────────────────┘
```

### 6.2 장비 스킬

장비를 장착하면 해당 스킬이 자동으로 활성화

#### 무기 스킬 (액티브)
장비마다 고유한 액티브 스킬 1개 보유

| 영혼의 단검 | 급소 찌르기 | ATK 250% + 크리 +30% |
| 성스러운 철퇴 | 심판의 일격 | ATK 200% + HP 10% 회복 |

## 8. 전투 시스템

### 8.1 자동 전투

방치형 게임에 맞춘 자동 전투 시스템

#### 전투 흐름
```
1. 몬스터 조우 (자동)
2. 기본 공격 (자동)
3. 스킬 사용 (쿨타임마다 자동)
4. 몬스터 처치
5. 보상 획득
6. 다음 몬스터 조우
```

#### 공격 속도
기본 공격 간격: 1초
SPD에 따라 감소 (최소 0.3초)

```
공격 간격 = 1초 / (1 + SPD/100)
```

### 8.2 데미지 계산

#### 물리 데미지
```
기본 = ATK * 스킬 배율
방어 적용 = 기본 * (100 / (100 + 적 DEF))
최종 = 방어 적용 * (1 + 크리티컬 보너스)
```

#### 마법 데미지
```
기본 = MAG * 스킬 배율
저항 적용 = 기본 * (100 / (100 + 적 MDF))
최종 = 저항 적용 * (1 + 크리티컬 보너스)
```

#### 크리티컬
```
기본 확률 = 5% + (LUK * 0.3%)
크리티컬 데미지 = 150% (기본)
```

### 8.3 생존 시스템

HP가 0이 되면 사망
사망 시:
- 현재 스테이지에서 부활
- 10초 무적 시간
- HP 50% 회복

#### 사망 페널티
| 조건 | 페널티 |
|------|--------|
| 권장 레벨 이하 스테이지 | 없음 |
| 권장 레벨 초과 스테이지 | 사냥 효율 50% 감소 (30초) |
| 보스전 사망 | 보스 도망 |

**설계 의도**: 권장 레벨 스테이지에서는 편하게 사냥, 상위 스테이지 도전 시에는 약간의 리스크

### 8.4 보스 전투

10분 사냥 후 스테이지 보스 등장

#### 보스 특징
- HP: 일반 몬스터의 50배
- ATK: 일반 몬스터의 3배
- 특수 스킬 사용
- 처치 시간 제한: 2분

#### 보스 전투 타이밍
```
0분 ~ 10분: 일반 몬스터 사냥
10분: 보스 등장 (2분 제한)
12분: 보스 처치 완료 or 실패
12분 ~ 22분: 일반 몬스터 사냥
22분: 다음 보스 등장
...
```

#### 보스 실패 시
- 보스 도망
- 다음 보스까지 10분 대기 (일반 사냥 계속)
- 페널티 없음

### 13.3 주간 보스

매주 특별 보스 등장

#### 주간 보스 스케줄
| 요일 | 보스 | 비고 |
|------|------|------|
| 월요일 00:00 | 보스 출현 | 주간 리셋, 새 보스 등장 |
| 일요일 23:59 | 보스 퇴장 | 도전 마감 |

#### 도전 규칙
- 주당 도전 횟수: 3회
- 보스 레벨: 유저 최고 레벨 기준 조정
- 제한 시간: 5분 (보스 체력 70% 이상 깎으면 클리어)

#### 보스 목록 (랜덤 순환)
| 보스 | 특성 | 추가 드롭 |
|------|------|-----------|
| 고대의 골렘 | 높은 방어력, 느린 공격 | 방어구 확률 증가 |
| 암흑의 용 | 광역 공격, 지속 피해 | 무기 확률 증가 |
| 타락한 정령왕 | 회복 방해, 디버프 | 장신구 확률 증가 |
| 혼돈의 마왕 | 균형형, 페이즈 변화 | 전체 드롭률 증가 |

#### 보상 구조
| 클리어 순위 | 보상 |
|-------------|------|
| 1회 클리어 | 영웅 장비 1개 (확정), 골드 10,000, 각성석 1 |
| 2회 클리어 | 영웅/전설 장비 (10% 전설), 골드 15,000, 각성석 1 |
| 3회 클리어 | 전설 장비 (30% 확률), 골드 20,000, 각성석 2 |

#### 미도전 시 보상
- 주간 보스를 도전하지 않으면 보상 없음
- 단, 다음 주로 도전 횟수 이월 불가

## 14. 밸런스 기준

### 14.1 가계부 재화 획득량 (일일)

적극적 사용자 기준 (일반 기록 10건 + 인증 5건)

| 재화 | 기본 보상 | 인증 보상 | 퀘스트 | 합계 |
|------|-----------|-----------|--------|------|
| 강화석 | 5~10 | 10~15 | 10~20 | 25~45 |
| 스킬북 | 0~2 | 1~2 | 1~2 | 2~6 |
| 가챠 티켓 | 0~1 | 0~1 | 0~1 | 0~3 |

### 14.2 게임 재화 획득량 (일일)

활성 사용자 기준 (하루 4시간 플레이)

| 재화 | 획득량 |
|------|--------|
| 골드 | 30,000~80,000 |
| 경험치 | 레벨 의존 |
| 장비 드롭 | 20~50개 |
| 강화석 (게임 내) | 3~5개 |

#### 골드 수입/지출 밸런스 (일일)
| 항목 | 수량 |
|------|------|
| 수입: 사냥 | +30,000~80,000 |
| 수입: 보스 | +5,000~15,000 |
| 지출: 강화 5회 | -20,000~50,000 |
| 지출: 부스터 | -5,000~10,000 |
| 지출: 상점 구매 | -10,000~20,000 |
| **순수익** | +0~15,000 |

**설계 의도**: 골드가 적당히 남아돌지만 과도하게 축적되지 않도록 조정

### 14.3 성장 속도 예상

| 목표 | 예상 소요 |
|------|-----------|
| 레벨 10 | 1~2일 |
| 레벨 30 | 1~2주 |
| 레벨 50 (전직) | 1~2개월 |
| 레벨 100 | 4~6개월 |
| +10 장비 | 2~3개월 |
| 전설 풀셋 | 3~6개월 |

### 14.4 강화석 소모량

+10 장비 1개 완성 예상 (평균 시도 횟수 포함)

| 단계 | 필요량 | 평균 시도 | 실제 소모량 | 누적 |
|------|--------|-----------|-------------|------|
| +1~+6 | 62개 | 1.3회 | 약 80개 | 80 |
| +7~+10 | 280개 (고급) | 2.5회 | 약 700개 (고급) | 780 (일반 환산) |

**일반 강화석 환산**: 고급 1개 = 일반 10개
**총 소요**: 일반 강화석 약 7,080개 또는 고급 강화석 약 280개 + 일반 80개

### 14.5 가계부 필요성

강화석 일일 소모 vs 획득 비교

| 활동 | 강화석 |
|------|--------|
| 일일 강화 시도 (5회) | -약 30개 |
| 가계부 없이 획득 | +3~5개 (게임 내 소량 드롭) |
| 가계부 기본 보상 | +25~45개 |

결론: 가계부 없이도 아주 느리게 성장 가능하나, 가계부 사용 시 5~10배 빠른 성장

### 14.6 고급 강화석 획득량 (일일)

| 획득처 | 일일 획득량 |
|--------|-------------|
| 영수증 인증 보상 | 1~3개 |
| 퀘스트 보상 | 2~5개 |
| 일반 강화석 교환 (10:1) | 사용자 선택 |
| 4시간 무료 상자 (운) | 0~2개 |
| 합계 | 3~10개 |

+10 강화 시 고급 강화석 약 280개 필요 → 약 1~2개월 소요

### 21.2 자동 정리

#### 자동 정리 규칙
- 등급별 분류: 일반 → 고급 → 희귀 → 영웅 → 전설 → 신화
- 장비 유형별 분류: 무기 → 방어구 → 장신구
- 최근 획득순 정렬 옵션

#### 필터링
| 필터 | 옵션 |
|------|------|
| 등급 | 전체, 일반~신화 개별 선택 |
| 장비 유형 | 무기, 방어구, 장신구 |
| 착용 가능 | 현재 레벨 이하 장비만 |
| 강화 상태 | 미강화, +1~+5, +6~+9, +10 이상 |

### 21.3 인벤토리 가득 참 처리

#### 알림
- 인벤토리 80% 도달 시 경고
- 100% 도달 시 자동 사냥 장비 드롭 중지 (경험치/골드는 지속 획득)

#### 권장 행동
1. 불필요 장비 분해
2. 상점 판매
3. 창고 이동 (향후 확장)
4. 인벤토리 확장
